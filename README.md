# Kanban

A simply front-end to-do application.

## Prerequisites

Node version 18.x.x

## Cloning the repository

```
git clone https://github.com/AntonioErdeljac/next13-trello.git
```

## Install packages

```
npm i
```

## Setup .env file

```
VITE_CLERK_PUBLISHABLE_KEY
VITE_UNSPLASH_ACCESS_KEY
```

## Start the app

```
npm run dev
```
